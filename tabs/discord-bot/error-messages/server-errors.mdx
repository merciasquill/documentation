## Failed to Fetch Sea of Thieves

<Badge color="red" icon="server">
  server-side
</Badge>
<Badge color="ghost">logged</Badge>

This is an internal service error caused when the Sea of Thieves API is unavailable or has returned an invalid response. This is generally caused by 3rd party maintenance or downtime outside of Mercia's Quill's control. If the error repeatedly persists, please [contact](https://merciasquill.com/support) a developer for further assistance.

## Missing Interaction Handler

<Badge color="red" icon="server">
  server-side
</Badge>
<Badge color="ghost">logged</Badge>

This is an internal service error caused when the associated action or handler function for an interaction could not be found. If the error repeatedly persists, please [contact](https://merciasquill.com/support) a developer for further assistance.

## Unexpected Server Error

<Badge color="red" icon="server">
  server-side
</Badge>
<Badge color="ghost">logged</Badge>

This is an internal service error caused for unexpected reasons. If the error repeatedly persists, please [contact](https://merciasquill.com/support) a developer for further assistance.
