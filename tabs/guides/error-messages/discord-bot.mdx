## Active Command Cooldown

<Badge color="yellow" icon="desktop">
  client-side
</Badge>
<Badge color="ghost">not logged</Badge>

This is a client-side error caused when recently using the same command in a short timeframe. Please try again later after the requested cooldown has ended.

<Info>
  Repeatedly using the same command in a short timeframe will exponentially
  increase rate limits.
</Info>

## Collector Timeout

<Badge color="yellow" icon="desktop">
  client-side
</Badge>
<Badge color="ghost">not logged</Badge>

This is a client-side error caused when a message does not receive a valid response within the given timeframe. Please try again, but select an option quicker.

## Failed to Fetch Sea of Thieves

<Badge color="red" icon="server">
  server-side
</Badge>
<Badge color="ghost">logged</Badge>

This is an internal service error caused when the Sea of Thieves API is unavailable or has returned an invalid response. This is generally caused by 3rd party maintenance or downtime outside of Mercia's Quill's control. If the error repeatedly persists, please [contact](https://merciasquill.com/support) a developer for further assistance.

## Invalid Autocomplete Option

<Badge color="yellow" icon="desktop">
  client-side
</Badge>
<Badge color="ghost">not logged</Badge>

This is a client-side error caused when a command option with autocompletion enabled is overridden and an invalid value is manually provided. Please try again, but select an autocompleted choice.

## Missing Cast Progress

<Badge color="yellow" icon="desktop">
  client-side
</Badge>
<Badge color="ghost">not logged</Badge>

This is a client-side error caused when no progress using the [/cast play](/tabs/discord-bot/available-commands/fun-commands#play-subcommand) subcommand could be found for the given timeframe. Please try again later after using the `/cast play` subcommand.

## Missing Interaction Handler

<Badge color="red" icon="server">
  server-side
</Badge>
<Badge color="ghost">logged</Badge>

This is an internal service error caused when the associated action or handler function for an interaction could not be found. If the error repeatedly persists, please [contact](https://merciasquill.com/support) a developer for further assistance.

## Unexpected Server Error

<Badge color="red" icon="server">
  server-side
</Badge>
<Badge color="ghost">logged</Badge>

This is an internal service error caused for unexpected reasons. If the error repeatedly persists, please [contact](https://merciasquill.com/support) a developer for further assistance.
