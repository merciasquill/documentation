### Active Command Cooldown

<Badge color="yellow" icon="desktop">
  client-side
</Badge>
<Badge color="ghost">not logged</Badge>

You have recently used this command and trigged a rate limit. Please wait until requested before using this command again. Note that repeatedly using the same command in a short time will gradually increase rate limit times.

### Missing Interaction Handler

<Badge color="red" icon="server">
  server-side
</Badge>
<Badge color="ghost">logged</Badge>

This is an internal service error caused when the associated action or handler function for an interaction could not be found. Please [contact](https://merciasquill.com/support) a developer for further assistance.
